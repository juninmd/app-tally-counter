# Tally Counter App

A simple and elegant Tally Counter application built with React Native and Expo.

## Features

- Multiple counters support
- Custom names for counters
- Increment/Decrement functionality
- Persistent storage (Coming soon)

## Getting Started

### Prerequisites

- Node.js
- pnpm

### Installation

```bash
pnpm install
```

### Running the App

```bash
pnpm start
```

## Tech Stack

- React Native
- Expo
- TypeScript
